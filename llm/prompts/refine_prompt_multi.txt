# lang=ja
以下のテキストは、文書から抽出された構造化テキストとOCRテキストです。
両方の内容は基本的に同じ文書から抽出されたものですが、抽出方法が異なります。

<structured_text>タグ内は文書の論理構造に基づいて抽出されたテキストです。
<ocr_text>タグ内は画像認識（OCR）によって抽出されたテキストです。

これらの内容を統合して、以下の要件に従って整形してください：

1. 両方の内容を比較し、より正確で完全な情報を統合する
2. 重複する内容は1つにまとめる
3. OCRエラーや文字化けがある場合は、構造化テキストの内容を優先する
4. 構造化テキストにない情報がOCRテキストにある場合は補完する
5. 日本語として自然で読みやすい文章に整形する
6. 元の文書の意味や構造を保持する
7. 不要な記号や改行は適切に整理する

# lang=en
The following text contains structured text and OCR text extracted from a document.
Both contents are basically extracted from the same document, but using different extraction methods.

The content within <structured_text> tags is text extracted based on the logical structure of the document.
The content within <ocr_text> tags is text extracted through image recognition (OCR).

Please integrate these contents and format them according to the following requirements:

1. Compare both contents and integrate more accurate and complete information
2. Merge duplicate content into one
3. If there are OCR errors or garbled characters, prioritize the structured text content
4. If there is information in the OCR text that is not in the structured text, supplement it
5. Format into natural and readable text
6. Preserve the meaning and structure of the original document
7. Properly organize unnecessary symbols and line breaks

{TEXT}
